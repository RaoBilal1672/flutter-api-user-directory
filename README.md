# User Directory App

A Flutter application developed as part of Week 4 of the Flutter Developers Internship Program.

The application fetches user data from a public REST API, parses JSON responses, and displays the information in a clean and user-friendly interface.

---

## 🚀 Features

- Fetch user data from a public API
- Parse JSON data into Dart objects
- Display users using ListView
- User Profile Screen
- Loading Indicator while fetching data
- Error Handling for failed API requests
- Clean and Responsive UI
- Organized Project Structure

---

## 📱 Screens

### Users Screen
Displays a list of users fetched from the API.

### Profile Screen
Displays detailed information about a selected user:
- Name
- Email
- Phone Number
- Website
